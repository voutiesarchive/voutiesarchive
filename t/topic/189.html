How will you react in a real life out-of-gas situation?

Few things in life go as planned or “by the book.” The same goes for diving when a stressful situation arrives, like running out of gas. Don’t believe us? Read some of these true stories about out of gas emergencies.

Τα περιστατικά με δύτη/δύτρια χωρίς αέρα μπορεί να διαφέρουν αρκετά μεταξύ τους. Συνήθως είναι καταστάσεις που δεν ομοιάζουν με τις περιγραφές των βιβλίων. Χρειάζεται να είμαστε νοητικά έτοιμοι για κάτι τέτοιο.

  • Θεωρήστε ότι οι δύτες χωρίς αέρα θα αμελήσουν να κάνουν τα ενδεδειγμένα σήματα με τα χέρια.
  • Έχετε το νου σας για σημάδια άγχους και να είστε προετοιμασμένοι να αντιδράσετε, μη διστάζετε.
  • Το να έχετε άφθονο διαθέσιμο αέριο για να μοιραστείτε σε τέτοιες καταστάσεις είναι πραγματικά σημαντικό.
  • Μπορείτε πάντα να επιλέξετε την ασφαλέστερη οδό σε ένα περιστατικό με δύτη χωρίς αέρα, δηλαδή να κάνετε κανονικά τις στάσεις ασφαλείας και όχι άμεση ανάδυση στην επιφάνεια, ακόμα κι αν αυτό δεν είναι η ταχύτερη οδός.
  • Θεωρήστε ότι τα περιστατικά χωρίς αέρα θα συμβούν στη χειρότερη δυνατή στιγμή/κατάσταση. Λάβετε κάτι τέτοιο υπόψιν στο καταδυτικό σας πλάνο, ενημερώστε τους συ-δύτες σας, μάθετε απέξω το πόσο αέριο χρειάζεστε για το εκάστοτε βάθος.
  • Κάντε εξάσκηση τις διαδικασίες αντιμετώπισης περιστατικών που μάθατε κατά τη διάρκεια της εκπαίδευσης. Και μετά κάντε λίγη εξάσκηση ακόμα.

Επίσης θα πρόσθετα:

  • Κάντε εξάσκηση τις διαδικασίες αντιμετώπισης περιστατικών. Αφού αποκτήσετε μια καλή εξοικείωση, αρχίστε να εξασκείστε τις ίδιες διαδικασίες σε ανύποπτο χρόνο προσπαθώντας να το κάνετε όσο πιο ρεαλιστικό γίνεται. Το τέλεια εκτελεσμένο s-drill όπου και οι δύο συμμετέχοντες είναι σε απόλυτη ετοιμότητα για την εκτέλεση της άσκησης, δεν σημαίνει ότι θα εξελιχθεί το ίδιο καλά σε καταστάσεις δύσκολες φορτισμένες με άγχος (και πιθανότατα πανικό).
